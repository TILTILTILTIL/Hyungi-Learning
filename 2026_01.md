이제 튜토리얼 따라하기 그만하고 기획에 따른 게임 제작을 시작하고 싶어서 작업 방식을 바꿈.  
아이디어의 구현에 필요한 기능들을 리스트업하고 그 기능들만 쏙쏙! 골라서 공부하고 구현해 나가는 방식.  
지금 내게 필요한 기능은?  
&nbsp;
* WASD 인풋을 받아 움직이는 시스템  
* 플레이어 눈높이에 카메라 고정시키고 마우스 움직임을 따라가게 만들기  
* Shift 누르면 달리게 하고 싶음  
* 눈앞에 있는 오브젝트와 좌클릭이나 키보드 eKey 등으로 상호작용이 가능하게끔 하고 싶음  
&nbsp;
이 중 1번부터 3번은 그냥 유니티 기본 제공 fps starter pack 설치해서 적용하면 끝나는 문제였음.  
상호작용을 하려면 Raycasting을 하게 해야 하는데, 우선 플레이어 마빡에 달린 카메라에서 광선 쏴서 맞는거 콘솔짱에 찍는것까지는 구현함.  
&nbsp;
    public class RayCast : MonoBehaviour
    {
        void Update()
        {
            RaycastHit hit;

            if (Physics.Raycast(Camera.main.transform.position, Camera.main.transform.forward, out hit, Mathf.Infinity))
            {
                Debug.Log(hit.collider.name); 
            }  
        }
    }
&nbsp;
RaycastHit 자료형 변수를 선언하고, Raycast 함수에서 out 키워드를 붙인 hit 매개변수로 정보를 반환하면 됨.  
&nbsp;
그 외 잡지식
* 플레이어 오브젝트가 Rigidbody 오브젝트들을 밀 수 있게 하려면 Basic RigidBody Push 컴포넌트에서 Can Push를 켜고 레이어를 지정해주면 된다. 깡통 캔 같은거 발로 차게 할 수 있을 듯. 축구공 같은 것도 만들 수 있고.  
* 플레이어가 총을 들게 하는 방법은 생각보다 간단하다. 그냥 총 모델 임포트하고 플레이어 오브젝트에 자식으로 붙이고 적당히 각도 조정해주면 됨. 그냥 자식으로 만들기보다 Empty object 부터 만들고 그 안에 model 엠티오브젝트 또 만들고 자식으로 붙여주면 유지보수하기 좋겠지.
